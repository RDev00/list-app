# Lista de paquetes usados en la app (usados)
  - http: Requests http para hacer fetch en nuestras APIs
  - url_launcher: Nos permite colocar urls (en formato a href) en nuestra app de manera comoda y sencilla
  - flutter_secure_storage: Este nos sirve como las cookies de navegador, lo que nos deja guardar tokens de acceso, modos (claro, oscuro), etc.
  - json_seriable: Hacer respuestas a json en flutter

## Nota
Actualmente no se usan unos, pero es debido a que no se han necesitado usar por lo que aun no se explica que hacen, pero apenas se usen empezaran a marcarse aqui.
