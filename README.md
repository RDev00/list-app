# [NAME]

## Descripción
Nuestra app se enfoca en permitir a usuarios de multiples plataformas darles el acceso a acceder a nuestra App, ya sea desde windows hasta android. Todo esto está creado con tecnologias que estamos descubriendo, así que en caso de que encuentres un bug puedes repotarlo en nuestra [sección de reportes]

## Lenguajes
Para la creación de esta app se usará el lenguaje de Dart, uno creado por Google para su tecnología.

## Tecnologías
La tecnología que usaremos será flutter, la cual nos permite crear apps multiplataforma de manera eficiente y sencilla, con un solo programador y un solo lenguaje.
